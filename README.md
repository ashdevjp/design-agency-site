# Design Agency Homepage

A modern design agency homepage built with Next.js 13+

## Live Demo
[View Live](your-vercel-link.vercel.app)

## Tech Stack
- Next.js 13 (App Router)
- Tailwind CSS
- React Icons
- Vercel (deployment)

## Setup
1. Clone the repo
2. Run npm install
3. Run npm run dev
4. Open localhost:3000

## Features
- Responsive mobile first design
- Hero, Services, Portfolio, Contact sections
- Hover animations on cards
- Contact form with validation
- SEO metadata
- Dark themed UI
