# Design Agency Homepage

A modern design agency homepage built with Next.js 13+

## Live Demo
[View Live](https://forxeldesign.vercel.app/)

## Tech Stack
- Next.js 13 App Router
- React 19
- React Icons and Lucide React for iconography
- Global CSS with responsive and animated UI
- ESLint for code quality
- Vercel deployment

## Setup
1. Clone the repo
2. Run npm install
3. Run npm run dev
4. Open localhost:3000

## Features
- Responsive, mobile-first homepage with sticky navigation
- Hero, Services, Portfolio, and Contact sections
- Animated background blobs and floating hero cards
- Filterable portfolio gallery with category buttons
- Contact form with client-side validation and send state
- Light/dark theme support
- Icon-powered service cards using `Lucide React` and `React Icons`
- Optimized portfolio images using Next.js `Image` component
- Clean modern UI with hover effects and animated transitions
